CMP ISBN Amanunensis

Last updated by Roedy Green ©1998-2001 Canadian Mind Products.

This Applet will help you tidy up ISBNs (International Standard Book Numbers). It tidies them into standard form with the dashes in the correct places (the rules where they go are quite complicated). It also validates the group number and the heck digit. Just paste the raw ISBN into the left window, click Tidy then copy the tidied ISBN from the right window.

I tidied ISBN looks like this: 0-9600688-8-0 or this 0-13-625666-X.

The amanuensis is also available with Java source to download. When you run ISBN Amanuensis as an application, it is even more automatic. You don't need to do any manual pasting and copying. Just click Tidy to convert the current contents of the clipboard. The tidied ISBN replaces the old the raw text clipboard contents. This automatic feature is not available in the Applet version because the Applet sandbox considers it a security risk.

This Applet requires one of:

  • Any version of Netscape 4.7 or 6.0 plus the Java Plug-in 1.3 (née Activator) from http://java.sun.com/products/plugin/.
  • Opera 5.02 with the Java Plug-in 1.3.
  • Microsoft Explorer 5.5 with the Java Plug-in 1.3.
  • This version will also work with Java powered, recent, smart non-Plug-in that know enough to bypass the plug-in HTML.
Click here for the native Java Version. If you don't have a JDK 1.1 capable browser you can still download the amanuensis and run it either as an application or an Applet.

If this Applet does not work for you with the Java Plug-In, there is a good chance that there is a problem with your Plug-In installation. You can test your Plug-In installation at Sun's Plug-In Test Page. Version 1.1 and 1.2 of the Plug-in often interfere with each other, so totally removing the plug-in, purging the plug-in directories and registry, and starting over sometimes works. Tweak the Java Plug-In Control panel, a separate application from your browser. If you are using Opera, see the notes in the Java & Internet Glossary on installing the Plug-In.
