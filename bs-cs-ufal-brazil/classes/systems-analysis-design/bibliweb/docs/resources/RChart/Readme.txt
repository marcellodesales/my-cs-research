RChart is java class library (and applet) that will add charts to your java
application or web site. In order to run the example application unzip the file .zip and read the HowToRun.txt file.

FILES:

- Examples.htm: applet examples
- Javadoc\ directory: javadoc files.
- Help_english.html: Parameters of the servlet and applet
- charapp.java: sample java application
- chartApplet.java: source code of the applet
- RChartServlet.java: source code of the servlet
- servlet_examples\ directory: examples for the servlet
- run*.bat: batchs file to run sample application with different Java VM.
- version.txt: version information
