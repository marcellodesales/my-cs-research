Algoritmo Caminhos mais Curtos - Método 
              O algoritmo opera pegando um conjunto S daqueles vértices pelos quais a menor distância do vértice 1 seja conhecida. Inicialmente, o vértice 1 é o único no conjunto S. A cada passo, adicionamos a S o vértice permanecente pelo qual o menor caminho do vértice 1 tenha sido determinado. O problema é determinar qual o vértice a adicionar a S em cada passo. Imaginemos os vértices já em S como tendo sido coloridos, e pensemos nas arestas sendo juntadas aos menores caminhos da fonte 1 àqueles vértices como também coloridos.
