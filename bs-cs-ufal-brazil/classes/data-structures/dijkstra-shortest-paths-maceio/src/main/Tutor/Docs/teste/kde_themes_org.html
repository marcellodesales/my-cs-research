Assimilated site theme and looking for staff - kde.t.o News
Posted by cjr on 2nd November 1999 @ 01:52 PM PDT
Some 20,000 people have already downloaded "Assimilated", but still it seemed like fun to me to port it to a site theme to give it some extra visibility. If you want the old site theme back, you can set your personal site preferences to the "KDE" site theme.

More important: KDE.themes.org is looking for staff.
More info can be found here or at #themes.org on openprojects. I'm away for a couple of days so you have to be patient.
